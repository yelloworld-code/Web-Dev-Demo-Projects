# Candy Museum

A themed website for a fictional candy museum, built with Bootstrap 4. Features a colorful, fun design showcasing various candy-related content.

## Features

- **Bootstrap Components**: Navigation, cards, carousels, and responsive grid
- **Themed Design**: Candy-inspired color scheme and imagery
- **Responsive Layout**: Mobile-friendly design
- **Interactive Elements**: Carousel and collapsible navigation
- **Custom Styling**: Additional CSS for theme-specific styling

## Technologies Used

- **Bootstrap 4**: CSS framework for responsive design
- **HTML5**: Semantic markup
- **CSS3**: Custom styling and theming
- **Google Fonts**: Roboto font family

## Getting Started

### Prerequisites

- A web browser

### Usage

Simply open `index.html` in your web browser to view the website.

## Project Structure

```
├── index.html      # Main HTML file
├── index.css       # Custom styles
└── img/           # Candy images and assets
```

## Bootstrap Features Used

- **Navigation**: Responsive navbar with toggle
- **Grid System**: Responsive columns and layout
- **Cards**: Content cards for different sections
- **Carousel**: Image carousel for featured content
- **Buttons**: Styled buttons and links
- **Typography**: Bootstrap typography classes
