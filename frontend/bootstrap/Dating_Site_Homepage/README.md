# Dating Website Homepage

A homepage design for a fictional dating website called "Purrfect Match", built with Bootstrap 4. Features a modern, attractive design for a dating platform.

## Features

- **Bootstrap Components**: Navigation, jumbotron, cards, forms, and responsive layout
- **Modern Design**: Clean, professional dating site interface
- **Responsive Layout**: Mobile-friendly design
- **Interactive Elements**: Collapsible navigation and forms
- **Custom Styling**: Theme-specific colors and styling

## Technologies Used

- **Bootstrap 4**: CSS framework for responsive design
- **HTML5**: Semantic markup
- **CSS3**: Custom styling and theming
- **Font Awesome**: Icon library
- **Google Fonts**: Lora font family

## Getting Started

### Prerequisites

- A web browser

### Usage

Simply open `index.html` in your web browser to view the homepage.

## Project Structure

```
├── index.html      # Main HTML file
└── index.css       # Custom styles
```

## Bootstrap Features Used

- **Navigation**: Dark navbar with responsive toggle
- **Jumbotron**: Hero section for main messaging
- **Grid System**: Responsive columns and layout
- **Cards**: Feature cards and content sections
- **Forms**: Contact and signup forms
- **Buttons**: Styled action buttons
