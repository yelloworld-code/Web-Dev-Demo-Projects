# Image Gallery

A simple, responsive image gallery built with Bootstrap 4. Features a clean grid layout for displaying photos.

## Features

- **Bootstrap Grid**: Responsive grid system for image layout
- **Clean Design**: Minimal, professional gallery interface
- **Responsive Layout**: Adapts to different screen sizes
- **Navigation**: Simple navbar with branding
- **Custom Styling**: Additional CSS for gallery-specific styling

## Technologies Used

- **Bootstrap 4**: CSS framework for responsive grid
- **HTML5**: Semantic markup
- **CSS3**: Custom styling

## Getting Started

### Prerequisites

- A web browser

### Usage

Simply open `index.html` in your web browser to view the gallery.

## Project Structure

```
├── index.html      # Main HTML file
└── index.css       # Custom styles
```

## Bootstrap Features Used

- **Grid System**: Responsive columns for image layout
- **Navigation**: Dark navbar with responsive toggle
- **Cards/Images**: Bootstrap image classes and responsive utilities
- **Container**: Bootstrap container for layout

## Learning Objectives

This project demonstrates:

- Bootstrap 4 grid system
- Responsive image galleries
- Basic Bootstrap components
- Static website development
- CSS layout techniques
